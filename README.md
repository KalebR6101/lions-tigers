# Dogs, JavaScript & An API üê∂

This project is a web application that fetches and displays dog images from a public API using JavaScript, Fetch, Promises, and Async/Await. 

It is based on Brad Schiff's tutorial: [Dogs, JavaScript & An API](https://www.bradtraversy.com/tutorials/dogs-js-api/)

## Features
- Fetches random dog images from a public API
- Displays images in a slide show
- Responsive and simple UI
